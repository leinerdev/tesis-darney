CONTROL DE NIVEL DE LÍQUIDO POR MEDIO DE VISIÓN ARTIFICIAL UTILIZANDO SISTEMA EMBEBIDO

Darney Rafael Aguas Manzano

Resumen

En la actualidad, la prioridad de muchas empresas se basa en la automatización de procesos, esto implica aumentos al nivel de producción ya sea de un producto o servicio, hace mucho mas eficiente la línea de operación en cuanto a maquinaria y de la materia prima a utilizar, por esta razón, el siguiente proyecto tiene como objetivo primordial el diseño e implementación de un sistema inteligente con visión artificial en la tarjeta embebida Raspberry pi4 para el control de llenado de tanque en un prototipo en el laboratorio de automatización y control de la universidad de pamplona ver (Figura 1), el sistema inteligente con visión artificial se apoya con inteligencia artificial a partir de una red neuronal entrenada en una maquina virtual configurada con cuadernos escritos en lenguaje python, con Google colab, el sistema se desarrollaró en tres fases principales, una fase principal que realiza la creación de una base de datos simétrica alimentada por imágenes, en segunda fase se realiza el preprocesamiento de imágenes junto a la aplicación de técnicas matemáticas, y la clasificación de imágenes. En la fase de creación de base de datos primeramente se recolectaron imágenes utilizando la cámara de la Raspberry pi4 junto al etiquetado de cada imagen con el tipo de nivel de líquido, en la fase de procesamiento de imágenes se aplicaron técnicas con el fin de destacar las zonas de interés y crear patrones característicos los cuales se utilizarán en la fase de clasificación de imágenes por medio de la creación y entrenamiento de un modelo de red neuronal que fue capaz de diferenciar los niveles de líquido con una alta precisión, además se muestra la detección del porcentaje de nivel para controlar una válvula de bola automáticamente con un nivel de liquido requerido por el usuario, a su vez se realizó el diseño e implementación de una interfaz gráfica programada en Python en sincronía con el modelo de red neuronal entrenado y la detección del porcentaje para controlar el nivel de llenado de tanques en el prototipo desde la tarjeta embebida raspberry pi4, por ultimo se desarrolló una pagina web tipo estatica programada desde cero utilizando Html, Css y JavaScript, en ella se encuentra la documentación que fue necesaria para la realización de este proyecto, teniendo en cuenta la necesidad de que personas accedan a esta información.

Objetivos

Objetivo General

Implementar un control de nivel de líquido por medio de visión artificial utilizando un sistema embebido.

Objetivos Específicos

  • Aplicar técnicas de procesamiento a las imágenes capturadas para entrenar múltiples redes neuronales para la clasificación del nivel de liquido.
  • Seleccionar la mejor red neuronal capaz de medir el porcentaje del nivel de liquido presente en el tanque del prototipo.
  • Implementar el modelo de red neuronal en la raspberry pi4 realizando captura de imágenes en tiempo real para determinar el porcentaje de nivel de liquido en el tanque.
  • Programar e implementar una interfaz gráfica en python que permita el control de nivel de liquido en los tanques.
