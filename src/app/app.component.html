CONTROL DE NIVEL DE LÍQUIDO POR MEDIO DE VISIÓN ARTIFICIAL UTILIZANDO SISTEMA EMBEBIDO

Darney Rafael Aguas Manzano

Resumen

El siguiente proyecto tiene como objetivo primordial diseñar e implementar un sistema inteligente con visión artificial en Raspberry pi4 para el control de llenado de tanques en un prototipo en el laboratorio de automatización y control de la universidad de pamplona, el sistema inteligente con visión artificial se apoyará con inteligencia artificial a partir de redes neuronales, el sistema se desarrollará en tres fases principales, creación de la base de datos, procesamiento de imágenes, clasificación de imágenes. En la fase de creación de base de datos se recolectarán imágenes a través de la cámara de la Raspberry pi4 junto al etiquetado de cada imagen con el tipo de nivel de líquido, en la fase de procesamiento de imágenes se aplicarán técnicas con el fin de destacar las zonas de interés y crear patrones característicos los cuales se utilizarán en la fase de clasificación de imágenes por medio de la creación y entrenamiento de un modelo de red neuronal capaz de diferenciar los niveles de líquido y finalmente el diseño e implementación de una interfaz gráfica programada en Python en sincronía con el modelo de red neuronal entrenado para controlar el nivel de llenado de tanques en el prototipo.

Objetivos

Objetivo General

Implementar un control de nivel de líquido por medio de visión artificial utilizando un sistema embebido.

Objetivos Específicos

  • Aplicar técnicas de procesamiento a las imágenes capturadas para entrenar múltiples redes neuronales para la clasificación del nivel de liquido.
  • Seleccionar la mejor red neuronal capaz de medir el porcentaje del nivel de liquido presente en el tanque del prototipo.
  • Implementar el modelo de red neuronal en la raspberry pi4 realizando captura de imágenes en tiempo real para determinar el porcentaje de nivel de liquido en el tanque.
  • Programar e implementar una interfaz gráfica en python que permita el control de nivel de liquido en los tanques.
